# Portfolio Website

A modern, responsive, and user-friendly portfolio website developed as a frontend assignment. The website is built using HTML5, CSS3, and JavaScript (ES6), focusing on clean design, accessibility, and an engaging user experience across all devices.

---

## Project Overview

This project was developed as an assignment provided by an organization. The objective was to design and develop a professional portfolio website that presents a designer's profile, skills, services, projects, and contact information in a visually appealing and organized manner.

The website follows modern UI/UX principles and is fully responsive with a clean layout, smooth navigation and dark mode support ensuring a seamless experience on desktops, tablets, and mobile devices.
